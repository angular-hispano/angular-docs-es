# Invalid Shadow DOM selector

El selector de un componente que usa `ViewEncapsulation.ShadowDom` no cumple los requisitos del nombre de etiqueta de elemento personalizado.

Para que un nombre de etiqueta se considere un nombre de elemento personalizado válido, debe:

- Estar en minúsculas.
- Contener un guión.
- Comenzar con una letra \(a-z\).

## Depurando el error

Renombra el selector de tu componente para que cumpla los requisitos.

**Antes:**

```ts
@Component({
  selector: 'comp',
  encapsulation: ViewEncapsulation.ShadowDom
…
})

```

**Después:**

```ts
@Component({
  selector: 'app-comp',
  encapsulation: ViewEncapsulation.ShadowDom
…
})
```
