# Missing Reference Target

<docs-video src="https://www.youtube.com/embed/fUSAg4kp2WQ"/>

Angular no puede encontrar una directiva con el nombre de exportación `{{ PLACEHOLDER }}`.
Esto es común cuando falta una importación o falta un [`exportAs`](api/core/Directive#exportAs) en una directiva.

ÚTIL: Este es el equivalente en el compilador de un error de tiempo de ejecución común [NG0301: Exportación no encontrada](errors/NG0301).

## Depurando el error

Usa el nombre de cadena de la exportación no encontrada para rastrear las plantillas o módulos que usan esta exportación.

Asegúrate de que todas las dependencias estén correctamente importadas y declaradas en nuestros módulos.
Por ejemplo, si la exportación no encontrada es `ngForm`, necesitaremos importar `FormsModule` y declararlo en nuestra lista de importaciones en `*.module.ts` para resolver el error de exportación faltante.

```ts

import { FormsModule } from '@angular/forms';

@NgModule({
…
imports: [
FormsModule,
…

```

Si recientemente agregaste una importación, necesitarás reiniciar tu servidor para ver estos cambios.
